The workflow:

  1. Connect to your Power BI Desktop model
  2. Analyze - 71 best practice rules run instantly
  3. Fix - AI agent reads your model, proposes a fix, waits for your OK
  4. Verify - rerun analysis to confirm

Built on Microsoft ecosystem:

  • GitHub Copilot SDK - enterprise AI agent
  • Power BI Modeling MCP Server
  • Tabular Editor 2 - BPA engine
  • Power BI Desktop
The architecture is simple: the app orchestrates Tabular Editor for analysis and the Copilot SDK for AI fixes. The MCP server is the bridge to the live semantic model. Everything is auditable.
All sessions persisted in SQLite for full auditability.
